# SystemVerilog I2C LED Driver
**ECE571 Final Project - Team 1**: Riley Cameron, Liukee Liu, Mohammad Alshaiji, Sal Esmaeil

### Getting Started
1. Open *QuestaSim* and select 'Open an existing project'
1. Navigate to the root directory of this repository and select the **ece_571_prj_team1.mpf** file
1. In the terminal run `do run.do` to run the simulation (run.do currently picks up **all** .sv files)
